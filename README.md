# vocabulary_program
건국대학교 자바프로그래밍 수업의 과제로 만든 단어장 프로젝트로, 단어장생성, 단어 뜻 출력, 단어 뜻 검색, 객관식 퀴즈, 오답노트, 단어추가 기능이 있는 영어 단어장입니다~! 위의 기능들을 수행하는 단어장을 어떤 화면 구성으로 만들어야 효과적일지 많이 생각할 수 있었던 프로젝트였습니다.
